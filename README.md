# 株価分析くん 📈

yfinance を使って日経225・グロース銘柄を分析し、  
「配当利回り × 底値圏（極小値）検出」でおすすめ銘柄を提示する Web アプリ。

---

## ファイル構成

```
kabuka/
├── app.py           # Flask バックエンド (メインサーバー)
├── analyzer.py      # スコアリング・極小値検出ロジック
├── stocks.py        # 銘柄マスターリスト
├── notifier.py      # メール通知
├── config.json      # SMTP 設定 (要編集)
├── requirements.txt # 依存ライブラリ
└── templates/
    └── index.html   # フロントエンド
```

---

## セットアップ

### 1. Python 環境 (3.10 以上推奨)

```bash
# 仮想環境を作成（推奨）
python -m venv venv
source venv/bin/activate      # Mac/Linux
# venv\Scripts\activate       # Windows

# ライブラリをインストール
pip install -r requirements.txt
```

### 2. 初回セットアップ（Chart.js をローカルに取得）

```bash
python setup.py
```

### 3. サーバー起動

```bash
python app.py
```

ブラウザで http://localhost:5000 を開く。

---

## 使い方

1. サイドバーで **指数（日経225 / グロース）** と **セクター** を選択
2. 「**銘柄スキャン開始**」ボタンを押す  
   → yfinance が前日終値・1年分の株価履歴・配当利回りを取得
3. **銘柄一覧タブ** で全銘柄を表示。ヘッダークリックでソート可能
4. **おすすめタブ** でスコア上位の銘柄をカード表示
5. 銘柄をクリックすると **チャートモーダル** が開き、極小値（底値候補）を確認できる

### スコアリング基準

| 項目 | 最大点 |
|------|--------|
| 配当利回り (5%以上=30点、4%=25点、3%=18点、2%=10点) | 30 |
| 底値圏検出（微分符号変化 + 直近60日極小値付近） | 35 |
| 長期下落→短期反発（底打ちパターン） | 15 |
| 52週安値5%以内 | 15 |
| **合計** | **100** |

### 買いシグナル判定

| スコア | シグナル |
|--------|----------|
| 60以上 | 今すぐ買い 🟢 |
| 40〜59 | 要注目 🟡 |
| 20〜39 | 様子見 ⚪ |
| 0〜19  | 見送り 🔴 |

## メール通知設定

`config.json` を編集してください。Gmail の場合は **アプリパスワード** を使用します。

```json
{
  "smtp_host": "smtp.gmail.com",
  "smtp_port": 587,
  "smtp_user": "your-email@gmail.com",
  "smtp_pass": "xxxx-xxxx-xxxx-xxxx"
}
```

Gmail アプリパスワードの取得:  
https://support.google.com/accounts/answer/185833

## データ取得について

- **データソース**: Yahoo Finance (yfinance ライブラリ経由)
- **取得タイミング**: ボタンを押したとき（1日1回程度を推奨）
- **キャッシュ**: 6時間キャッシュ（再スキャンで強制更新）
- **対象期間**: 直近1年の終値データ
